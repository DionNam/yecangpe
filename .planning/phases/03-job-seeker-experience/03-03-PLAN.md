    1. Create `apps/web/components/jobs/like-button.tsx`:
       - 'use client'
       - Import useOptimistic, useTransition from react
       - Import Heart icon from lucide-react
       - Import Button from @/components/ui/button
       - Import toggleLike from @/app/actions/likes

       Props interface:
       - postId: string
       - initialLiked: boolean
       - initialCount: number
       - canLike: boolean (false for employers)

       Implementation:
       - const [isPending, startTransition] = useTransition()
       - const [optimisticState, setOptimisticState] = useOptimistic(
           { liked: initialLiked, count: initialCount },
           (current, newLiked: boolean) => ({
             liked: newLiked,
             count: newLiked ? current.count + 1 : current.count - 1
           })
         )
       - handleClick function:
         - If !canLike, return early (or show tooltip)
         - const newLikedState = !optimisticState.liked
         - startTransition(async () => {
             setOptimisticState(newLikedState)
             await toggleLike(postId)
           })
       - Render Button with:
         - variant="ghost"
         - onClick={handleClick}
         - disabled={!canLike || isPending}
         - Heart icon: filled red if liked, outline gray if not
         - Display optimisticState.count next to heart
       - If !canLike, show heart without click handler (view only)

    2. Update `apps/web/components/jobs/job-detail.tsx`:
       - Import LikeButton
       - Replace placeholder with actual LikeButton component
       - Pass: postId, isLiked, displayLikes (as initialCount), canLike

    3. Verify `apps/web/app/(main)/jobs/[id]/page.tsx` passes all required props:
       - isLiked from user_liked_post RPC
       - displayLikes from getDisplayMetrics
       - canLike from seeker profile check
  </action>
