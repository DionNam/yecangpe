**Part A: apps/web/lib/validations/newsletter.ts**

Create Zod validation schema:

```typescript
import { z } from 'zod'

export const newsletterSchema = z.object({
  email: z.string().email('Please enter a valid email address'),
  name: z.string().min(1, 'Name is required').max(100, 'Name is too long'),
  type: z.enum(['job_seeker', 'employer'], {
    message: 'Please select a subscription type'
  })
})

export type NewsletterFormData = z.infer<typeof newsletterSchema>
```

**Part B: apps/web/app/actions/newsletter.ts**

Create server action for newsletter subscription:

1. 'use server' directive
2. Import createClient from '@repo/supabase/server'
3. Import newsletterSchema from '@/lib/validations/newsletter'
4. Export async function subscribeNewsletter(formData: FormData)
5. Parse formData: email, name, type
6. Validate with newsletterSchema.safeParse
7. On validation error: return { error: result.error.errors[0].message }
8. Insert into newsletter_subscribers table via supabase
9. Handle error code '23505' (unique constraint) -> return { error: 'This email is already subscribed' }
10. Handle other errors -> return { error: 'Failed to subscribe. Please try again.' }
11. On success -> return { success: true }
