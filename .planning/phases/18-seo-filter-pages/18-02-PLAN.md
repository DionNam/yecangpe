  <files>apps/web/app/(main)/jobs/by-type/[type]/page.tsx</files>

Create the job type filter page route. This is a server component with ISR.

**Structure:**
```
export const revalidate = 300  // 5 min ISR

export async function generateStaticParams()
export async function generateMetadata({ params })
export default async function JobsByTypePage({ params })
```

**generateStaticParams:**
- Import JOB_TYPES from '@repo/lib'
- Return `JOB_TYPES.map(({ code }) => ({ type: code }))`

**generateMetadata:**
- Await params to get `type` string
- Import and use `generateFilterPageMetadata('by-type', type)` from '@/lib/filter-page-data'
- Also query Supabase for job count to include in description: `supabase.from('job_posts').select('*', { count: 'exact', head: true }).eq('review_status', 'published').eq('hiring_status', 'hiring').eq('job_type', type)`
- Pass jobCount to generateFilterPageMetadata if it accepts it

**Page component:**
- Await params to get `type` string
- Validate type against JOB_TYPES - if not found, call `notFound()` from 'next/navigation'
- Create Supabase client via `createClient()` from '@repo/supabase/server'
- Check if user is authenticated (for job card heart/share buttons): `supabase.auth.getUser()`
- Query jobs: `supabase.from('job_posts').select('*', { count: 'exact' }).eq('review_status', 'published').eq('hiring_status', 'hiring').eq('job_type', type).order('published_at', { ascending: false }).limit(20)`
- Query company count: `supabase.from('job_posts').select('company_name', { count: 'exact', head: true }).eq('review_status', 'published').eq('hiring_status', 'hiring').eq('job_type', type)` (or use a distinct query if possible, otherwise just use job count)
- Get filter page data: `getFilterPageTitle('by-type', type)`, `getFilterPageDescription('by-type', type)`, `getFilterPageFAQs('by-type', type)`, `getFilterCrossLinks('by-type', type)`
- Build ItemList JSON-LD schema (same pattern as existing /jobs page):
  ```typescript
  const baseUrl = process.env.NEXT_PUBLIC_SITE_URL || 'https://hanguljobs.com'
  const itemListSchema = {
    '@context': 'https://schema.org',
    '@type': 'ItemList',
    name: title,
    itemListElement: jobs.map((job, index) => ({
      '@type': 'ListItem',
      position: index + 1,
      item: {
        '@type': 'JobPosting',
        name: job.title,
        url: `${baseUrl}/jobs/${job.slug}`,
        hiringOrganization: { '@type': 'Organization', name: job.company_name },
        datePosted: job.published_at || job.created_at,
      },
    })),
  }
  ```
- Render FilterPageLayout with:
  - title/titleKo/description from getFilterPageTitle/Description
  - jobCount from query count
  - companyCount (approximate: just use count or derive from jobs array unique company_name count)
  - faqs from getFilterPageFAQs
  - crossLinks from getFilterCrossLinks
  - filterName from the JOB_TYPES.find().name
  - children: ItemList JSON-LD script + JobListTable component (reuse from '@/components/jobs/job-list-table') passing jobs and isAuthenticated

**Important:**
- Do NOT use searchParams in this page (kills static generation)
- Type the params as `Promise<{ type: string }>` (Next.js 15 async params)
- Import Database type from '@repo/supabase/types' for job post typing
