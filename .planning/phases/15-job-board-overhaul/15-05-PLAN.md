    Completely rewrite `job-card.tsx` to display all PRD-mandated fields. This component is used for mobile layout.

    **Imports:**
    ```typescript
    import { JOB_TYPES, KOREAN_LEVELS, ENGLISH_LEVELS } from '@repo/lib'
    import { ShareButton } from './share-button'
    import { Badge } from '@/components/ui/badge'
    import { MapPin, Calendar, DollarSign } from 'lucide-react'
    import { formatDistanceToNow } from 'date-fns'
    import { ko } from 'date-fns/locale'
    import Image from 'next/image'
    ```

    **Props interface update:**
    Remove `displayViews` and `displayLikes` props - the PRD card design doesn't show view/like counts. Keep `job`, `isAuthenticated`. Add no new props.

    **Layout structure:**
    ```
    [Logo/Initial] [Title + Company]        [New badge]
                   [Location with MapPin]
                   [Salary with DollarSign]
                   [Badges: job_type, location_type, korean, english]
                   [Date] .................. [Heart] [Share]
    ```

    **Implementation details:**

    1. **Company Logo / Initial fallback:**
       - If `job.company_logo_url` exists, show with `Image` component (48x48, rounded, object-contain with p-1)
       - If `job.image_url` exists (current image field), use that as fallback
       - Otherwise show first letter of `company_name` in a colored circle (bg-slate-100, text-slate-400)

    2. **"New" badge:**
       - Show red/destructive badge with "New" text if `published_at` is within last 7 days
       - Use simple date comparison: `new Date(published_at) > new Date(Date.now() - 7 * 24 * 60 * 60 * 1000)`

    3. **Location display:**
       - MapPin icon + location text
       - Remote: "원격근무", Hybrid: "하이브리드", On-site: country name from `getCountryName()`

    4. **Salary display (conditional):**
       - Only show if `salary_min` or `salary_max` exists
       - Format: "₩2,000,000 - ₩3,000,000 / 월" (use currency symbol from SALARY_CURRENCIES + period from SALARY_PERIODS)
       - Import `SALARY_CURRENCIES` for symbol lookup
       - Use `.toLocaleString()` for number formatting

    5. **Badge row:**
       - Job type badge: **blue** (`bg-blue-600 text-white`) with `nameKo` from JOB_TYPES
       - Work location type badge: **emerald** (`bg-emerald-600 text-white`)
       - Korean level badge: **outline** with "한국어: {nameKo}" (only if set and not 'not_specified')
       - English level badge: **outline** with "영어: {nameKo}" (only if set and not 'not_specified')

    6. **Footer row:**
       - Left: Calendar icon + relative date via `formatDistanceToNow(new Date(published_at), { addSuffix: true, locale: ko })`
       - Right: Share button (reuse existing `ShareButton` component)
       - Heart/like button: Keep reference but don't block on it (existing `like-button.tsx` can be wired later)

    7. **Click handler:** Keep existing logic (authenticated -> navigate to detail, not authenticated -> show login modal)

    **Color decisions from Phase 13:**
    - job_type: blue
    - work_location_type: emerald
    - country: slate
