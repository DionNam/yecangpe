**Severity Levels:**

- **High:** Major feature not working, significant user impact, workaround difficult

- **Low:** Minor issue, cosmetic problem, minimal user impact

**Status Values:**
- **Open:** Bug discovered, not yet fixed
- **In Progress:** Developer working on fix
