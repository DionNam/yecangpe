**1. Create `apps/web/app/actions/job-alerts.ts`** - Server actions for job alert CRUD:

```typescript
'use server'

import { createClient } from '@repo/supabase/server'
import { revalidatePath } from 'next/cache'
```

Three server actions:

- `createJobAlert(data: { keywords?: string, country?: string, job_type?: string, frequency: string })`:
  - Authenticate user
  - Insert into `job_alerts` table: `{ user_id: user.id, keywords, country, job_type, frequency, is_active: true }`
  - Use `(supabase as any)` for the query (job_alerts may not be in generated types)
  - `revalidatePath('/dashboard')`
  - Return `{ success: true }` or `{ error: string }`

- `updateJobAlert(id: string, data: { keywords?: string, country?: string, job_type?: string, frequency?: string, is_active?: boolean })`:
  - Authenticate user
  - Update: `supabase.from('job_alerts').update(data).eq('id', id).eq('user_id', user.id)` (RLS also enforces this, but belt-and-suspenders)
  - `revalidatePath('/dashboard')`
  - Return `{ success: true }` or `{ error: string }`

- `deleteJobAlert(id: string)`:
  - Authenticate user
  - Delete: `supabase.from('job_alerts').delete().eq('id', id).eq('user_id', user.id)`
  - `revalidatePath('/dashboard')`
  - Return `{ success: true }` or `{ error: string }`
