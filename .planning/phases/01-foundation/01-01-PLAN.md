    -- Helper functions (SECURITY DEFINER):
    - is_admin() - checks if user.role = 'admin'
    - is_employer() - checks if employer_profiles exists for user
    - is_seeker() - checks if seeker_profiles exists for user

    -- Policies for each table:
    USERS:
    - Read own data (authenticated)
    - Admin read all (authenticated + is_admin)
    - Update own data (authenticated)

    SEEKER_PROFILES:
    - Read own (authenticated)
    - Admin read all (authenticated + is_admin)
    - Insert own (authenticated)
    - Update own (authenticated)

    EMPLOYER_PROFILES:
    - Read own (authenticated)
    - Admin read all (authenticated + is_admin)
    - Insert own (authenticated)
    - Update own (authenticated)

    JOB_POSTS:
    - Anyone read published (anon, authenticated)
    - Authors read own any status (authenticated)
    - Admin read all (authenticated + is_admin)
    - Employers create (authenticated + is_employer)
    - Admin create (authenticated + is_admin)
    - Authors update pending (authenticated)
    - Authors update hiring_status of published (authenticated)
    - Admin update all (authenticated + is_admin)
    - Admin delete (authenticated + is_admin)

    LIKES:
    - Read own (authenticated)
    - Seekers create (authenticated + is_seeker)
    - Delete own (authenticated)

    GLOBAL_METRICS_CONFIG:
    - Admin read (authenticated + is_admin)
    - Admin update (authenticated + is_admin)
