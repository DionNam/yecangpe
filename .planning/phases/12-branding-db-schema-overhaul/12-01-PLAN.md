Add unique constraint on slug:
```sql
DO $$
BEGIN
  IF NOT EXISTS (SELECT 1 FROM pg_constraint WHERE conname = 'job_posts_slug_key') THEN
    ALTER TABLE job_posts ADD CONSTRAINT job_posts_slug_key UNIQUE (slug);
  END IF;
END $$;
```

**Migration 2: New tables (20260207_create_new_tables.sql)**

Create job_alerts table:
- id UUID PRIMARY KEY DEFAULT gen_random_uuid()
- user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE
- keywords TEXT (nullable)
- country TEXT (nullable)
- job_type TEXT (nullable)
- frequency TEXT NOT NULL CHECK (frequency IN ('instant', 'daily', 'weekly'))
- is_active BOOLEAN NOT NULL DEFAULT true
- created_at TIMESTAMPTZ DEFAULT now()
- updated_at TIMESTAMPTZ DEFAULT now()

RLS on job_alerts:
- Users can SELECT/INSERT/UPDATE/DELETE own alerts (auth.uid() = user_id)
- Admins can ALL (is_admin())

Create newsletter_subscribers table:
- id UUID PRIMARY KEY DEFAULT gen_random_uuid()
- email TEXT NOT NULL UNIQUE
- name TEXT (nullable)
- type TEXT NOT NULL CHECK (type IN ('job_seeker', 'employer'))
- is_active BOOLEAN NOT NULL DEFAULT true
- created_at TIMESTAMPTZ DEFAULT now()

RLS on newsletter_subscribers:
- Anyone can INSERT (WITH CHECK true)
- Users can SELECT own by email match
- Users can UPDATE own by email match
- Admins can ALL (is_admin())

Add indexes:
- idx_job_alerts_user_id ON job_alerts(user_id)
- idx_job_alerts_active ON job_alerts(is_active) WHERE is_active = true
- idx_newsletter_email ON newsletter_subscribers(email)
- idx_job_posts_slug ON job_posts(slug)
- idx_job_posts_job_type ON job_posts(job_type)
- idx_job_posts_status ON job_posts(status)

Also save the SQL files to the migrations directory for version control.
