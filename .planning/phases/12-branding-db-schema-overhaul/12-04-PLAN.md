  <action>
1. Download Pretendard variable font file. Use the variable font (single file) instead of multiple static weights for smaller bundle:
   ```bash
   mkdir -p apps/web/app/fonts
   curl -L -o apps/web/app/fonts/PretendardVariable.woff2 "https://github.com/orioncactus/pretendard/raw/main/packages/pretendard/dist/web/variable/woff2/PretendardVariable.woff2"
   ```
   If curl fails, try the CDN: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/woff2/PretendardVariable.woff2`

2. Update apps/web/app/layout.tsx to use next/font/local:

   Add at top of file:
   ```typescript
   import localFont from 'next/font/local'

   const pretendard = localFont({
     src: './fonts/PretendardVariable.woff2',
     variable: '--font-pretendard',
     display: 'swap',
     weight: '100 900',  // Variable font supports all weights
   })
   ```

   Update the html tag to apply the font:
   ```tsx
   <html lang="ko" className={pretendard.variable}>
   ```

   Keep the existing `<head>` contents (Google Analytics scripts) unchanged.
   Keep the existing body contents (Suspense, SiteHeaderWrapper, ScrollFix) unchanged.

Note: Using variable font (single file ~3.3MB) instead of 4 static files because:
- Simpler configuration
- Supports any weight value (not just 400/500/600/700)
- next/font optimizes and subsets automatically
  </action>
  <verify>
Verify font file exists: `ls -la apps/web/app/fonts/PretendardVariable.woff2` — file should exist and be >1MB.
Verify layout.tsx has pretendard import and className application.
  </verify>
  <done>Pretendard variable font file exists in apps/web/app/fonts/. layout.tsx imports and applies the font via next/font/local with --font-pretendard CSS variable.</done>
