## 창업 아이템 개요(요약)

| 항목 | 내용 |
|------|------|
| **명칭** | HangulJobs (한글잡스) |

### 아이템 개요

한류(K-pop, K-drama 등)의 글로벌 확산으로 한국어를 학습하는 외국인이 폭발적으로 증가하고 있으나(TOPIK 응시자 4년간 2배 증가, 42만 8천명/2024), 이들 한국어 구사 글로벌 인재와 해외 진출 한국 기업을 효과적으로 연결하는 전문 플랫폼이 부재합니다.

**HangulJobs**는 한국어 능력 수준별 매칭, 직종별 필터링, 20개 산업 카테고리 지원을 통해 한국어를 구사하는 전 세계 구직자와 한국어 인재가 필요한 기업을 직접 연결하는 채용 플랫폼입니다.

**핵심 기능**: 한국어/영어 능력 수준별 필터링, AI 기반 인재-공고 매칭, 180개국 지원, 이중언어(한/영) UI, 구직자 프로필 공개 시스템, 기업 인재 탐색 기능

**고객 제공 혜택**:
- **구직자**: 한국어 능력을 활용한 커리어 기회를 한 곳에서 탐색
- **기업**: 한국어 구사 외국인 인재를 빠르고 정확하게 채용

---

### 문제 인식 (Problem) 요약

1. 한국어 학습자 급증(TOPIK 42.8만명, 듀오링고 1,100만명)에도 이들의 취업을 연결하는 전문 플랫폼 부재
2. 한국 기업 해외 투자 $639.5억(2024) 규모이나, 현지에서 한국어 구사 인재 채용 경로가 없음
3. 기존 플랫폼(잡코리아, 사람인)은 국내 한국인 대상, LinkedIn은 한국어 능력 필터 미지원

### 실현 가능성 (Solution) 요약

- 이미 MVP(hanguljobs.com) 개발 완료 및 운영 중 (Next.js 15 + Supabase 기반)
- 구직자/기업 양면 플랫폼 구축 완료, 관리자 대시보드 운영 중
- 협약기간 내 AI 매칭 고도화, 유료 모델 도입, 마케팅 확대 계획

### 성장전략 (Scale-up) 요약

- 비즈니스 모델: 기업 유료 공고 게시($129~/건, NihongoJobs 벤치마크) + 프리미엄 인재 탐색 구독
- 1차 타겟: 미국, 일본, 베트남, 인도네시아 (TOPIK 상위국)
- 3년 내 SOM $1M-$5M 목표

### 팀 구성 (Team) 요약

- (대표자 역량 기입 필요)

# 1. 문제 인식 (Problem) — 창업 아이템의 필요성

## 1-1. 한국어 학습자 폭발적 증가

ㅇ **한류(Hallyu)가 만든 한국어 학습 열풍**

- TOPIK(한국어능력시험) 응시자: 218,869명(2020) → **428,585명(2024)**, 4년간 **2배 증가** (역대 최다)
  - 87개국 323개 지역에서 시험 실시 (출처: [TOPIK 공식](https://www.topik.go.kr/))
- 세종학당 수강생: 101,675명(2020) → **239,020명(2025)**, **2.4배 증가**
  - 87개국 252개 학당 운영 (출처: [Korea Times, 2026.02](https://www.koreatimes.co.kr/lifestyle/trends/20260225/sejong-institutes-overseas-enrollment-tops-230000-for-first-time))
- 듀오링고 한국어 학습자: **1,100만명 이상**, 세계 7위 학습 언어
  - 브라질, 프랑스, 독일, 인도, 멕시코에서 **가장 빠르게 성장하는 언어** (출처: [Duolingo 2024 Language Report](https://blog.duolingo.com/2024-duolingo-language-report/))
- 글로벌 한류 팬: **2.25억명** (10년간 24배 증가) (출처: [KoreaTechDesk](https://koreatechdesk.com/k-content-investment-critical-window))

ㅇ **한류 콘텐츠 산업의 경제적 영향**

- 한국 콘텐츠 산업 수출: **$132억(2022, 역대 최고)**, 연평균 10% 성장
  - (출처: [Korea Herald](https://www.koreaherald.com/article/10011173))
- 콘텐츠 수출 $1억 당 관련 소비재 수출 **$1.8억** 유발 효과
- 정부 콘텐츠 산업 투자: 5,039억원(2021) → **1.74조원(2024)**, 3.5배 증가
  - (출처: [KOCCA](https://www.kocca.kr/en/main.do))

## 1-2. 한국 기업의 글로벌 확장과 인재 수요

ㅇ **해외 진출 한국 기업의 급증**

- 한국 해외직접투자(OFDI): **$639.5억(2024)**
  - 유럽 +25.5%, 아시아 +9.7%, 중동 +176.9% 증가 (출처: [기재부](https://english.moef.go.kr/pc/selectTbPressCenterDtl.do?boardCd=N0001&seq=6118))
- IT·통신 분야 투자: $30.6억, **전년 대비 73.9% 성장**
- 삼성전자 해외 직원: **137,000명** (국내 125,000명 초과) (출처: [Samsung Sustainability Report](https://www.samsung.com/global/sustainability/))
- 연간 약 **3,000개** 한국 기업이 해외 법인 설립

ㅇ **한국어 구사 인재에 대한 수요 급증**

- 해외 법인 운영 시 본사와의 소통을 위해 한국어 능력 필수
- 한국 문화·비즈니스 관행을 이해하는 현지 인재 확보 난항
- 세종학당 수료생 중 22.4%가 해외 한국 기업 취업, 16.2%가 한국 내 한국 기업 취업 → **취업 연결 니즈 입증**

## 1-3. 기존 솔루션의 한계와 문제점

ㅇ **한국어 인재 채용을 위한 전문 플랫폼 부재**

| 기존 플랫폼 | 한계점 |
|-------------|--------|
| 잡코리아/사람인 | 한국어 전용, 외국인 인재 기능 없음, 국내 채용만 지원 |
| LinkedIn | 한국어 능력 필터 미지원, 한국어 인재 특화 검색 불가 |
| WorknPlay | 한국 내 영어 교육 일자리에 특화, 전문직 미지원 |
| KOWORK | 한국 내 외국인 취업만 지원, 글로벌 매칭 불가 |
| 서울글로벌센터 | 서울 한정, 기본적인 채용 정보만 제공 |

(출처: [WorknPlay](https://www.worknplay.co.kr/), [KOWORK](https://kowork.kr/en), [Linguasia](https://linguasia.com/jobs-korea))

ㅇ **일본어 시장과의 비교: NihongoJobs.com의 성공 사례**

- 일본어 채용 시장에는 **NihongoJobs.com**이 존재
  - 미국 내 일본어-영어 이중언어 인재 매칭 플랫폼
  - 공고 게시 $129~/건, 약 2,500개 활성 공고, 20,000명+ 사용자 (출처: [NihongoJobs](https://nihongojobs.com/))
- **한국어 시장은 일본어보다 더 빠르게 성장 중이나, 동일한 서비스가 존재하지 않음**
  - 듀오링고에서 한국어는 5개 주요국에서 "가장 빠르게 성장하는 언어"
  - 일본어 시장 대비 한류의 글로벌 도달 범위가 더 넓음 (2.25억 팬)

ㅇ **우리의 직접 경험에서 확인한 페인포인트**

- 인도네시아에서 한국인 클라이언트 대상 개발사 운영
- IT 배경 + 한국어 구사 인력 채용의 극심한 어려움 직접 체험
- 반대로, 한국어를 잘하는 현지 인재들도 한국 기업을 찾기 어려워함
- **양측 모두 니즈가 있으나 연결 통로가 부재한 것이 핵심 문제**

## 1-4. 인구구조 변화에 따른 외국인 인재 필요성 증대

ㅇ **한국의 인구 위기**

- 합계출산율 0.7 미만 → 세계 최저 수준
- 경제활동 비참여 인구 **50만명 돌파**(2025.02) (출처: [Korea Times](https://www.koreatimes.co.kr/southkorea/globalcommunity/20250417/new-visa-aiming-to-address-labor-shortage-faces-fierce-backlash))
- 생산가능인구 감소 → 외국인 노동자 구조적 필요성 증가

ㅇ **정부의 외국인 인재 유치 정책 확대**

- 디지털 노마드 비자(F-1-D) 도입: 2024.01~ (최대 2년 체류) (출처: [EY](https://www.ey.com/en_gl/technical/tax-alerts/south-korea-announces-new-pilot-program-for-digital-nomad-visas))
- E-7-4 숙련기능인력 비자 쿼터: **35,000명**(2025) (출처: [Korea Herald](https://www.koreaherald.com/article/10011795))
- E-9 고용허가제 쿼터: **130,000명**(2025)
- 한국어 요건 완화: 비자 취득 후 2년 내 한국어 충족 가능(2026년까지)
- 수도권 비자 시스템: 지방자치단체별 맞춤형 비자 허용 (출처: [Korea.net](https://www.korea.net/NewsFocus/policies/view?articleId=266947))

# 2. 실현 가능성 (Solution) — 창업 아이템의 개발 계획

## 2-1. 현재 개발 완료 현황 (MVP)

ㅇ **이미 작동하는 플랫폼 보유 (hanguljobs.com)**

- **기술 스택**: Next.js 15 (App Router) + Supabase (PostgreSQL) + Tailwind CSS v4
- **모노레포 구조**: 사용자 웹(web) + 관리자 대시보드(admin) + 공유 라이브러리(lib)
- **주요 구현 기능**:

| 구분 | 구현 완료 기능 |
|------|----------------|
| 구직자 | 채용공고 검색/필터링(한국어 수준, 직종, 근무형태, 국가별), 관심공고 저장, 프로필 생성, 직접 지원 |
| 기업 | 무료 채용공고 게시, 리치텍스트 에디터, 조회/관심/지원 통계, 인재 탐색 |
| 관리자 | 공고 승인/반려, 사용자 관리, 사이트 설정 |
| 공통 | Google OAuth 인증, 한/영 이중언어 UI, 180개국 지원, 20개 산업 카테고리, PostgreSQL 전문검색(FTS), 뉴스레터 |

## 2-2. 협약기간 내 개발 계획

ㅇ **1단계 (1~4개월): AI 매칭 고도화 & 유료 모델 구축**

- AI 기반 인재-공고 자동 매칭 알고리즘 개발 (한국어 수준 + 직종 + 경력 가중치)
- 기업용 유료 공고 게시 시스템 (결제 연동: Stripe)
- 프리미엄 인재 탐색 기능 (고급 필터, 인재 추천)
- TOPIK 점수 연동 기능 (인증된 한국어 능력 표시)

ㅇ **2단계 (5~8개월): 사용자 확대 & 수익화**

- 기업용 대시보드 고도화 (지원자 관리, 면접 스케줄링)
- 구직자 프로필 고도화 (포트폴리오, 경력 증명서 업로드)
- SEO 최적화 및 소셜미디어 자동 공유 기능
- 다국어 확장 (베트남어, 인도네시아어, 일본어 UI 추가)
