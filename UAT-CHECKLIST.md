## 1. Public Pages - 메인 (TC-001 ~ TC-006)

### TC-001: 메인 페이지 히어로 섹션
- **Flow:** `/` 접속
- **Check:**
  - [ ] "한국어 가능한 인재를 위한 글로벌 채용" 타이틀 렌더링
  - [ ] "구직자로 시작하기" 버튼 → `/job-seekers` 이동
  - [ ] "채용 공고 올리기" 버튼 → `/employers` 이동
  - [ ] 오른쪽 히어로 이미지 정상 로딩
  - [ ] 신뢰 배지 3개 (검증된 채용 공고, 글로벌 기회, 한국어 특화) 표시

### TC-002: 메인 페이지 통계 섹션
- **Flow:** `/` 스크롤 다운
- **Check:**
  - [ ] Active Jobs 숫자 > 0 표시
  - [ ] Companies 숫자 > 0 표시
  - [ ] Job Seekers 숫자 > 0 표시 (member_count_offset 반영)

### TC-003: 메인 페이지 최신 공고 섹션
- **Flow:** `/` 최신 공고 섹션
- **Check:**
  - [ ] 최신 공고 최대 6개 카드 표시
  - [ ] 각 카드: 제목, 회사명, 고용형태, 근무형태 뱃지
  - [ ] 카드 클릭 → 해당 공고 상세 페이지 이동
  - [ ] "전체 공고 보기" 링크 → `/jobs` 이동

### TC-004: 메인 페이지 검색바
- **Flow:** `/` 검색 섹션
- **Check:**
  - [ ] 직무명 키워드 입력 + 지역 입력 → 검색 버튼 클릭
  - [ ] `/jobs` 페이지로 이동, 검색 파라미터 반영
  - [ ] 인기 검색어 (번역, 교육, IT/엔지니어링, 마케팅, 원격근무) 클릭 동작

### TC-005: 메인 페이지 FAQ 아코디언
- **Flow:** `/` FAQ 섹션
- **Check:**
  - [ ] 5개 FAQ 항목 표시
  - [ ] 각 항목 클릭 → 답변 토글 (열기/닫기)
  - [ ] 동시에 여러 항목 열리는지/닫히는지 확인

### TC-006: 메인 페이지 HangulJobs 이용 방법 섹션
- **Flow:** `/` 이용 방법 카드
- **Check:**
  - [ ] "구직자 이용 방법" 카드 → `/job-seekers` 이동
  - [ ] "고용주 이용 방법" 카드 → `/employers` 이동

## 3. 공고 목록 & 검색 (TC-013 ~ TC-025)

- **Flow:** `/jobs` 접속
- **Check:**
  - [ ] 공고 카드 10개 단위 표시
  - [ ] 각 카드: 회사 이니셜 아바타, 제목, 회사명, 위치, 고용형태 뱃지, 한국어 수준
  - [ ] 게시일 표시 ("19일 전" 등 상대 시간)
  - [ ] "공유" 버튼 각 카드에 존재

### TC-014: 페이지네이션
- **Flow:** `/jobs` 하단 페이지네이션
- **Check:**
  - [ ] 페이지 번호 클릭 → 해당 페이지 공고 로딩
  - [ ] "Next" 버튼 → 다음 페이지 이동
  - [ ] URL에 `?page=2` 쿼리 파라미터 반영
  - [ ] 마지막 페이지에서 "Next" 비활성화

### TC-015: 키워드 검색
- **Flow:** `/jobs` 검색창에 "통역" 입력
- **Check:**
  - [ ] 디바운스 300ms 후 검색 실행
  - [ ] "통역" 포함 공고만 필터링
  - [ ] 검색어 클리어 시 전체 공고 복원
  - [ ] URL에 `?q=통역` 반영

### TC-016: 필터 - 고용 형태 (다중 선택)
- **Flow:** "고용 형태" 필터 드롭다운
- **Check:**
  - [ ] 6개 옵션 표시: 정규직, 파트타임, 계약직, 프리랜서, 인턴십, 일시적
  - [ ] "정규직" 선택 → 정규직 공고만 표시
  - [ ] "정규직" + "파트타임" 동시 선택 → 두 유형 모두 표시
  - [ ] URL: `?job_type=full_time,part_time`
  - [ ] 선택 해제 → 전체 복원

### TC-017: 필터 - 근무 형태 (다중 선택)
- **Flow:** "근무 형태" 필터 드롭다운
- **Check:**
  - [ ] 3개 옵션: 원격, 하이브리드, 현장
  - [ ] "원격" 선택 → 원격 공고만 표시
  - [ ] 복수 선택 → OR 조건 동작
  - [ ] URL: `?location_type=remote`

### TC-018: 필터 - 카테고리 (단일 선택)
- **Flow:** "카테고리" 필터 드롭다운
- **Check:**
  - [ ] 20개 카테고리 옵션 전부 표시 (번역, 교육, IT/엔지니어링, 마케팅, 디자인, 영업, 고객서비스, 재무, HR, 운영, 콘텐츠, 컨설팅, 의료, 서비스업, 제조, 법률, 미디어, 연구, 무역, 기타)
  - [ ] 카테고리 선택 → 해당 카테고리 공고만 표시
  - [ ] URL: `?category=translation`

### TC-019: 필터 - 근무 국가 (단일 선택)
- **Flow:** "근무 국가" 필터 드롭다운

  - [ ] 국가 선택 → 해당 국가 공고만 표시

### TC-020: 필터 - 한국어 수준 (단일 선택)
