# 창업 아이템 개요(요약)

| 항목 | 내용 |
|------|------|
| **명칭** | HangulJobs (한글잡스) |
| **범주** | 글로벌 한국어 인재 채용 매칭 플랫폼 (HR-Tech SaaS) |

### 아이템 개요

한류의 글로벌 확산으로 한국어 학습자가 폭발적으로 증가(TOPIK 42.8만명, 듀오링고 1,100만명)하고, 한국 기업의 해외 진출도 가속화($639.5억 OFDI)되고 있으나, **한국어 구사 외국인 인재와 한국 기업을 연결하는 전문 플랫폼이 부재**합니다.

자체 운영 커뮤니티(2,500명) 내 HR 45명 대상 설문조사 결과, **"적합한 후보자 찾기 어려움(38%)"**, **"감으로 한국어 실력 판단(56%)"**, **"비공식 채널 의존(31%)"** 등 구조적 문제가 확인되었습니다.

**HangulJobs**는 한국어 인재 전용 채용 플랫폼(180개국, 20개 산업 카테고리)을 기반으로, 3가지 AI 핵심 기능으로 이 문제를 해결합니다:

**핵심 솔루션**:
1. **동영상 자기소개 + AI 한국어 능력 자동 점수화** (STT + 발음/유창성 분석)
2. CV/이력서 기반 경력 데이터화 (AI 파싱 → 정밀 검색)
3. AI 기반 기업-인재 자동 매칭 (한국어 수준 + 직무 + 경력 가중치)

**고객 제공 혜택**:
- **구직자**: 한국어 능력이 객관적으로 검증되어 더 많은 기회 획득
- **기업**: 검증된 한국어 인재를 빠르고 정확하게 매칭, 채용 시간 40% 단축

### 문제 인식 (Problem)

1. 한국어 인재 채용 시 **"적합한 후보자 찾기 어려움" 38%** (HR 45명 설문)
2. 한국어 실력 검증 수단 부재 — **56%가 "감으로 판단"**
3. **31%가 지인/추천** 등 비공식 채널에 의존 → 전문 플랫폼 부재의 방증
4. 기존 플랫폼(잡코리아, LinkedIn)은 한국어 능력 필터 미지원

### 실현 가능성 (Solution)

- MVP(hanguljobs.com) **운영 중** + 인재 커뮤니티 **2,500명** + 5개국 **일일 크롤링** 가동
- 협약기간 내 **동영상 AI 한국어 검증** + **AI 매칭** + 유료 모델 구축
- 대표 TreeSoop/Asimula 개발사·스타트업 경험 + 공동창업자 개발사 5년

### 성장전략 (Scale-up)

- **초기 무료 → 단계적 유료 전환** (인재풀/기업 확보 후 프리미엄 기능부터 유료화)
- 수익원: 공고 게시($99/건) + AI 한국어 평가($29/건) + 인재 탐색 구독($299/월)
- 1차 타겟: 미국, 일본, 베트남, 인도네시아 (TOPIK 상위국)
- 3년 내 SOM ~$750K 목표

### 팀 구성 (Team)

- 대표: TreeSoop(AI/VR 개발사), Asimula(VR K-pop 스타트업), RnD(인도네시아 개발사) 운영 경험
- 공동창업자: 소프트웨어 개발사 5년 운영 경험
- **인도네시아 한국어 인재 카카오톡 커뮤니티 2,500명** 직접 운영 → 초기 사용자 확보 채널 보유

# 1. 문제 인식 (Problem) — 창업 아이템의 필요성

## 1-1. 창업배경 & 문제점

### (1) 창업배경 — 폭발하는 한국어 시장, 그러나 연결 통로가 없다

ㅇ **한류가 만든 글로벌 한국어 학습 열풍**

한류(K-pop, K-drama)의 글로벌 확산으로 한국어 학습자가 폭발적으로 증가하고 있습니다.

| 지표 | 수치 | 성장률 |
|------|------|--------|
| TOPIK 응시자 (2024) | **428,585명** (87개국 323개 지역) | 4년간 2배 증가 |
| 듀오링고 한국어 학습자 | **1,100만명+** | 5개 주요국 최다 성장 언어 |
| 세종학당 수강생 (2025) | **239,020명** (87개국 252개 학당) | 5년간 2.4배 증가 |
| 글로벌 한류 팬 | **2.25억명** | 10년간 24배 증가 |

출처: [TOPIK 공식](https://www.topik.go.kr/), [Duolingo 2024 Report](https://blog.duolingo.com/2024-duolingo-language-report/), [Korea Times](https://www.koreatimes.co.kr/lifestyle/trends/20260225/sejong-institutes-overseas-enrollment-tops-230000-for-first-time), [KoreaTechDesk](https://koreatechdesk.com/k-content-investment-critical-window)

ㅇ **한국 기업의 해외 진출 가속화 → 현지 한국어 인재 필요**

| 지표 | 수치 |
|------|------|
| 한국 해외직접투자 (2024) | **$639.5억** |
| IT·통신 분야 투자 성장 | **전년 대비 +73.9%** |
| 연간 신규 해외 법인 설립 | ~**3,000개** |
| 삼성전자 해외 직원 | **137,000명** (국내 125,000명 초과) |

출처: [기획재정부](https://english.moef.go.kr/pc/selectTbPressCenterDtl.do?boardCd=N0001&seq=6118), [Samsung Sustainability Report](https://www.samsung.com/global/sustainability/)

한국어를 배우는 외국인은 급증하고, 한국 기업의 해외 진출도 가속화되고 있으나, **한국어 구사 외국인 인재와 한국 기업을 연결하는 전문 채용 플랫폼은 존재하지 않습니다.**

### (2) 내부적 배경 — 대표자의 직접 경험에서 발견한 Pain Point

대표자는 인도네시아에서 소프트웨어 개발사(TreeSoop, RnD)를 운영하며 한국 클라이언트 대상 개발 프로젝트를 수행했습니다. 이 과정에서 **"한국어 + IT 역량을 동시에 갖춘 현지 인재"를 채용하는 것이 극도로 어렵다**는 문제를 직접 체험했습니다.

- 한국어를 잘하는 현지 인재는 **어디서 찾아야 하는지 알 수 없었고**
- 반대로, 한국어를 배운 현지 인재들도 **한국 기업의 채용 정보를 찾기 어려워했습니다**

이 양측의 니즈를 연결하기 위해 인도네시아 한국어 인재 카카오톡 커뮤니티를 개설했고, 현재 **2,500명 규모**로 성장했습니다. 이 커뮤니티 운영을 통해 한국어 인재 채용 시장의 구조적 문제를 깊이 파악하게 되었습니다.

### (3) 1차 시장검증 — HR 대상 설문조사 결과 (N=45)

커뮤니티 내에서 최근 1년 내 한국어 가능 외국인을 채용한 HR 및 기업 담당자 **45명**을 대상으로 설문조사를 실시했습니다.

**Q1. 채용 시 가장 어려웠던 점은?**

| 응답 | 비율 |
|------|------|
| **적합한 후보자 찾기 어려움** | **38%** |
| 실무 역량 검증의 어려움 | 22% |
| 한국어 실력 검증의 어려움 | 18% |
| 커뮤니케이션/문화 차이 | 12% |
| 비자 및 행정 문제 | 10% |

> 핵심 인사이트: **"사람이 없는 게 아니라, 검증된 인재를 찾기 어려운 것이 핵심"**

**Q2. 지원자의 한국어 실력을 어떻게 검증하나요?**

| 응답 | 비율 |
|------|------|

| 별도 테스트 진행 | 13% |
| 추천/레퍼런스 기반 | 11% |

> 핵심 인사이트: **"대부분 감으로 뽑고 있음 → 표준화된 한국어 검증 시스템이 부재"**

**Q3. 채용 시 가장 중요하게 보는 요소는?**

| 응답 | 비율 |
|------|------|
| **직무 관련 경험** | **34%** |
| 한국어 커뮤니케이션 능력 | 29% |
| 태도 및 문화 적합성 | 21% |
| 학력 및 자격증 | 9% |
| 기타 | 7% |

> 핵심 인사이트: **"한국어만 잘해서는 부족 → 직무 경험 + 한국어 능력을 함께 매칭해야 함"** — AI 매칭의 근거

**Q4. 채용 후 가장 크게 느낀 문제는?**

| 응답 | 비율 |
|------|------|

> 핵심 인사이트: **"채용 후에도 한국어 실력 미스매치 발생 → 사전 검증이 더 정밀해야 함"** — 동영상 AI 검증의 근거

**Q5. 채용 채널은 어디를 사용했나요?**

| 응답 | 비율 |

> 핵심 인사이트: **"아직도 비공식 채널 의존도가 높음 — 전문 플랫폼 부재의 방증"**

**Q6. 이 시장에서 가장 필요하다고 느끼는 것은?**

| 응답 | 비율 |
|------|------|

> 핵심 인사이트: **"검증된 인재 풀(36%) + 평가 시스템(25%) + 빠른 매칭(21%) = 82%가 플랫폼 기능을 원함"**

## 1-2. 목표 시장 (고객 분석)

### (1) 시장 규모 (TAM/SAM/SOM) + 시장 동향

| 구분 | 규모 | 산출 근거 |
|------|------|-----------|
| **TAM** | ~$100M/년 | 해외 진출 한국 기업 8만+ x 평균 $650~$2,000/년 |
| **SAM** | ~$13M/년 | 동남아+일본+CIS 한국 기업 ~15,000개 x $650/년 + 구독 |
| **SOM (3년차)** | **~$750K/년** | 동남아 5개국 한국 기업 ~680개 전환 x $650/년 + 구독 매출 ~$300K |

## 2-2. 핵심 솔루션 — 설문 데이터 기반 3가지 AI 해결 방안

1장의 설문조사에서 발견한 Pain Point에 대응하는 솔루션을 설계했습니다.

HangulJobs는 한국어 구사 외국인과 한국 기업만을 위한 **전용 채용 플랫폼**(180개국, 20개 산업 카테고리, 이중언어 UI)을 기반으로, 다음 **3가지 AI 핵심 기능**으로 기존 채용 시장의 구조적 문제를 해결합니다.

### 솔루션 (1): 동영상 자기소개 + AI 한국어 능력 자동 점수화

> **해결하는 문제**: Q2 "인터뷰로 감으로 판단 56%" → **표준화된 한국어 검증 부재**

- 구직자가 **한국어 자기소개 영상**을 업로드 (1~3분)
- AI가 음성을 STT(Speech-to-Text) 변환 후 **발음 정확도, 유창성, 문법, 어휘력** 자동 분석
- **한국어 능력 점수(HangulScore)** 자동 산출 → 기업이 객관적으로 비교 가능
- 벤치마크: HireVue (40개 언어 AI 평가, CEFR 수준 평가 정확도 95%)

### 솔루션 (2): CV/이력서 기반 경력 데이터화

> **해결하는 문제**: Q1 "적합한 후보자 찾기 어려움 38%" → **인재 검색/필터 부족**

- 구직자 이력서를 AI가 자동 파싱 → 직무/경력/스킬을 **구조화된 데이터**로 변환
- 기업이 "한국어 중급 이상 + IT 경력 3년 + 인도네시아" 같은 조건으로 **정밀 검색** 가능
- 키워드 검색의 한계 극복: 의미 기반(Semantic) 매칭으로 40~60% 더 많은 적합 후보 발굴

### 솔루션 (3): AI 기반 기업-인재 자동 매칭

> **해결하는 문제**: Q6 "빠른 매칭 시스템 필요 21%" → **수동 검색 비효율**

- 공고 등록 시 AI가 **한국어 수준 + 직무 경력 + 근무 조건** 기반으로 적합 인재 자동 추천
- 기업에 Match Score(매칭 점수) 제공 → 채용 의사결정 시간 단축
- 구직자에게도 맞춤 공고 추천 → 지원율 향상
- 벤치마크: AI 매칭 도입 시 채용 소요 시간 **40% 단축**, 사전 검토 시간 **50% 절감** (업계 평균)

- 전환 경로: E-9/H-2 5년+ 보유자 → E-7-4 전환 가능
- 출처: [Korea Herald](https://www.koreaherald.com/article/10011795), [Korea.net](https://www.korea.net/NewsFocus/policies/view?articleId=266947), [KOWORK](https://kowork.kr/en/blog/e7-visa-2025-salary-requirements-en)
